About project

This small project is made just for fun. It is not too difficult but some new techniques were acquired. Arduino and similar devices are also not used so this little project will be atypical on this page. In comparison to the Solar project, the Highcharts diagram was used instead of CanvasJS. Data is acquired using API from CryptoCompare, for a personal (non-commercial) project. Data is sent to the server every five minutes, which then sends data to SQL database. It would be fun to use WebSockets and receive data in the application directly from the internet, but for now, this will work. This party will take place later. Data is then shown in diagrams on this page and it is updated every minute. Only BTC/EUR, BTC/USD, ETH/EUR, and ETH/USD charts are shown and this is more than enough to understand how this works.
