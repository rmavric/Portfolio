GPS project


Idea

The main goal of this small project was to learn how the GPS module works, save data that the GPS module gives, and show this data on the map. GPS is a system that gives information about the geographical position in the way that receivers communicate with a minimum of 4 satellites and calculate geographical latitude and longitude (altitude and time). Since I have to drive through the county very often because of work, it is ideal to turn on this small device and collect some data. Data is collected every second and gets saved to microSD card. To draw those routes on the map leaflet.js was used (the thing I saw for the first time, but it will be used in some future projects to explore more capabilities).

Components used

Arduino Nano V3.0 -> used to control NEO-6M GPS module and MicroSD module

NEO-6M GPS Module -> gives information about longitude and latitude through serial communication to arduino

MicroSD Module -> used for saving data to microSD card

MB102 Breadboard Power Supply or PowerBank (2A output) -> used to power the whole project
