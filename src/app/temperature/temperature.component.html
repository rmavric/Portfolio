Temperature project


Idea

This is a project that is not too demanding but it is practical. A small device that monitors the percentage of humidity and ambient temperature was built in this project and of course, those measurements are sent to the database, and then they are shown on this page. Measurements are taken once per hour using DHT11 Humidity & Temperature Sensor. It measures temperatures from 0° - 50° C and humidity percentages from 0% - 90%. To measure humidity it uses two electrodes that are near to each other. When the humidity in ambient is higher, more humidity comes between those electrodes, and resistance between them gets smaller. To measure temperature it uses a thermistor which changes its resistance, to be more precise, when the temperature gets higher resistance decreases. DHT11 is connected to WEMOS D1 mini, which is awake for 1 minute, sends measured data to the database, and then goes to sleep for about 59 minutes. RTC clock was not used, but that would be very useful to have more precise reading intervals because the internal clock inside Wemos is not too precise.

Components used

WEMOS D1 mini -> with ESP8266

DHT11 -> temperature and humidity sensor

Resistor 10 kΩ -> between 5V and data pin

MB102 Breadboard Power Supply -> used to power Wemos D1 mini and DHT11
